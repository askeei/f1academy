== Description ==

하나 최근글 위젯 플러그인은 하나보드 워드프레스 게시판 뿐 아니라 블로그 포스트, Product, 최근댓글 등 모든 포스트타입의 위젯을 자유롭게 사용할 수 있으며, 쇼트코드로 원하는 페이지에 삽입 가능합니다.

* [Hana Wordpress 커뮤니티](http://hanawordpress.com)
* [Hana Wordpress 스토어](http://hanawordpress.com/store)
* [플러그인 소개](http://hanawordpress.com/hana-like)
* [Hana Board 소개](http://hanawordpress.com/hana-board)
* [제작요청 / 기술지원](http://hanawordpress.com/technical-support)

= Features:  =

* 워드프레스 포스트, 우커머스 상품 뿐 아니라 게시판 게시물 등 모든 포스트 타입에 대해 사용 가능합니다.
* 하나보드 게시판과 간단하게 연동하고,사이드바, 푸터 위젯 뿐 아니라 모든 페이지에 편리하게 삽입 가능합니다.
* 게시물 뿐 아니라 최근 댓글까지 깔끔한 디자인을 제공합니다.
* 다양한 스킨을 제공하며, 인기글, 관련글 등 여러 기능이 업데이트 될 예정입니다.

== Frequently Asked Questions ==

= 원하는 스킨 또는 기능이 없을 땐 어떻게 해야 할까요? =
하나워드프레스 공식 홈페이지에서 문의 주시면 제작해 드립니다.

= 이 플러그인을 상업적으로 사용해도 될까요? =
개인, 프리랜서, 기업 관계없이 누구나 무료로 사용하실 수 있으며, 코드의 수정이 있을 경우 재배포는 불가능합니다.
