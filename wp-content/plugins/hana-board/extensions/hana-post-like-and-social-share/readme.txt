하나 추천&소셜공유 플러그인은 뛰어난 확장성과 안정성을 가진 한국형 워드프레스 게시판 플러그인입니다.

== Description ==

하나 추천&소셜공유 플러그인은 워드프레스의 플러그인 형태로 제공되는 무료 소프트웨어로 포스트 추천(Like)과 소셜 공유(Social Share)를 모두 사용하실 수 있습니다. 게시판과 블로그 포스트, 우커머스 상품, 포트폴리오 등 모든 포스트 타입에서 사용 가능합니다.

* [Hana Wordpress 커뮤니티](http://hanawordpress.com)
* [Hana Wordpress 스토어](http://hanawordpress.com/store)
* [플러그인 소개](http://hanawordpress.com/hana-like)
* [Hana Board 소개](http://hanawordpress.com/hana-board)
* [제작요청 / 기술지원](http://hanawordpress.com/technical-support)

= Features:  =

* 워드프레스 포스트, 우커머스 상품 뿐 아니라 게시판 게시물 등 모든 포스트 타입에서 사용 가능합니다.
* 하나보드 게시판과 간단하게 연동하고, 게시판 별 세부 설정이 가능합니다.
* 다양한 SNS가 꾸준히 업데이트 될 예정입니다.
* 편리한 관리자 UI와 다양한 옵션을 제공합니다.
