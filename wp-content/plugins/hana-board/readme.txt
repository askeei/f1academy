하나보드 워드프레스 게시판은 뛰어난 확장성과 안정성을 가진 한국형 워드프레스 게시판 플러그인입니다.

== Description ==

하나보드 워드프레스 게시판은 워드프레스의 플러그인 형태로 제공되는 무료 소프트웨어로 게시판, 갤러리, 웹진, 소셜공유, 게시물 추천, 게시판 위젯, 비밀글 등 다양한 기능을 제공합니다. 검색엔진최적화(SEO)되어 있으며, 워드프레스의 CORE에 충실하게 제작되었습니다. 기능 추가 및 스킨 제작 문의, 워드프레스 질문 등은 하나워드프레스 커뮤니티에서 받습니다.

= Features:  =

* 다양한 반응형 스킨들이 기본으로 제공됩니다.
* 누구나 쉽게 스킨을 제작하고 수정할 수 있습니다.
* 검색엔진최적화(SEO)가 되어 있어 네이버, 구글 등에 노출이 잘 됩니다.
* myCRED 등의 포인트 시스템과 호환성이 높습니다.
* 게시물 추천, 소셜공유, 위젯 등 다양한 확장 플러그인을 무료로 제공하며, 꾸준히 업데이트 됩니다.
* 편리한 관리자 UI와 다양한 옵션을 제공합니다.
* 대부분의 브라우저에서 원터치 멀티 파일 업로드 기능을 제공하여 편리하게 사진을 업로드할 수 있습니다.
